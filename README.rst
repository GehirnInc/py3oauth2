py3oauth2
==========
|travis|_ |coveralls|_ |pypibadge|_

-----
About
-----
This is a JSON Web Token library for Python 3 developed at `Gehirn Inc`_.

-------
Install
-------

.. code:: shell

  $ pip install py3oauth2

Or

.. code:: shell

  $ python setup.py install
